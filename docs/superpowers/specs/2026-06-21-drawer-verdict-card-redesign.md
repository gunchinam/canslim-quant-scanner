# 드로워 판단 카드 UI 재설계

## 배경

기존에 추가한 `dp-verdict-word` / `dp-timing-word` (큰 글씨 중앙 정렬 텍스트)가 드로워 기존 디자인과 이질감이 있었다. 카드 기반 디자인 언어와 맞지 않는 덩그러니 떠있는 텍스트 형태였으며, 설명도 기술적 용어 위주라 이해하기 어려웠다.

## 목표

- 드로워를 열자마자 "사세요 / 기다리세요 / 보류예요" 판단이 포스터급으로 바로 들어올 것
- 기존 한줄평 포스터와 동일한 디자인 언어 사용 (그라디언트 카드)
- 설명 문구는 비투자자도 이해할 수 있는 친절한 구어체 한국어

## 확정 디자인

### 살까? 탭

**판단 포스터 (풀사이즈 그라디언트 카드)**

```
┌─────────────────────────────────────┐
│ 이 종목, 지금 살까?        (eyebrow) │  ← 11px 상단 레이블
│                                     │
│ 기다리세요                (big word) │  ← 34px bold
│                                     │
│ 좋은 종목이지만 지금 당장 사기엔    │  ← 13px 3줄 구어체 설명
│ 타이밍이 살짝 애매해요.             │
│ 가격이 조금 내려올 때 사요.         │
│                        [배경 워터마크]│
└─────────────────────────────────────┘
```

색상 (conv 기준):
- conv ≥ 72: 초록 그라디언트 (#34D399 → #059669), 워터마크 "사요"
- conv 42–71: 노랑 그라디언트 (#FBBF24 → #D97706), 워터마크 "기다려"
- conv < 42: 빨강 그라디언트 (#F87171 → #DC2626), 워터마크 "보류"

**이유 목록** (포스터 아래)

4축 분석 결과 기반. ✅/⚠️ 아이콘 + 구어체 한 줄 설명. 예:
- ✅ 추세가 강해요 — 이평선이 나란히 올라가고 있어요
- ✅ 거래량이 받쳐줘요 — 사람들이 많이 사고 있어요
- ⚠️ 요즘 등락이 좀 심해요 — 갑자기 흔들릴 수 있어요

**한줄평 보조 카드** (이유 목록 아래)

기존 `dp-fa-haiku` 포스터(그라디언트)를 테두리 카드로 격하. 텍스트와 서브 설명만 표시.

---

### 언제? 탭

**미니 판단 카드** (살까? 탭 포스터의 축소 버전)

```
┌───────────────────────────────┐
│ 지금 타이밍은?    (eyebrow)   │  ← 10px
│ 🟡 조금 기다려요  (word)      │  ← 22px bold
│ 추가 확인 필요 · 확신도 70%   │  ← 12px
│                  [배경 워터마크]│
└───────────────────────────────┘
```

같은 conv 기준 색상 사용 → 살까? 탭과 항상 일치.

미니 카드 아래: 기존 종합 점수 + 4축 분석 (변경 없음)

## 구어체 설명 문구 (conv 구간별)

| 구간 | big word | 설명 3줄 |
|------|----------|----------|
| conv ≥ 72 | 사세요 | "지금이 딱 좋은 타이밍이에요. / 추세도 강하고 거래량도 받쳐줘요. / 분할로 들어가 보세요." |
| conv 42–71 | 기다리세요 | "좋은 종목이지만 지금 당장 사기엔 / 타이밍이 살짝 애매해요. / 가격이 조금 내려올 때까지 지켜봐요." |
| conv < 42 | 보류예요 | "지금은 사기 어려운 상황이에요. / 여러 지표가 안 좋은 신호를 보내고 있어요. / 관심 목록에만 담아두세요." |

## 이유 목록 문구 (4축 기반 구어체)

| 조건 | 아이콘 | 텍스트 |
|------|--------|--------|
| trend score ≥ 4 | ✅ | 추세가 강해요 — 이평선이 나란히 올라가고 있어요 |
| trend score ≤ 2 | ⚠️ | 추세가 꺾이고 있어요 — 방향이 불안정해요 |
| mom score ≥ 4 | ✅ | 가격 탄력이 좋아요 — 오르는 힘이 있어요 |
| mom score ≤ 2 | ⚠️ | 모멘텀이 약해요 — 오르는 힘이 부족해요 |
| vol score ≥ 4 | ✅ | 변동성이 낮아요 — 안정적으로 움직이고 있어요 |
| vol score ≤ 2 | ⚠️ | 요즘 등락이 좀 심해요 — 갑자기 흔들릴 수 있어요 |
| volm score ≥ 4 | ✅ | 거래량이 받쳐줘요 — 사람들이 많이 사고 있어요 |
| volm score ≤ 2 | ⚠️ | 거래량이 많이 빠졌어요 — 관심이 줄어들고 있어요 |
| bear_div 있음 (uptrend) | ⚠️ | 단기 과열 신호가 있어요 — 눌림목 올 수 있어요 |

점수가 3인 항목은 표시하지 않음 (중립).

## 변경 범위

### scanner.html
- `dp-verdict-word` div 제거
- `dp-timing-word` div 제거
- `dp-section-conclusion` 안에 `dp-verdict-poster` div 추가 (포스터 자리)
- `dp-fa-haiku` 클래스를 포스터에서 보조 카드로 변경
- `dp-section-timing` 안에 `dp-timing-mini` div 추가 (미니 카드)

### app.js
- `_renderEntryVerdict()`: verdict-word/timing-word 렌더 → 판단 포스터 + 미니 카드 렌더로 교체
- `_buildReasonItems()`: 4축 점수 기반 구어체 이유 생성 함수 추가
- 기존 `dp-fa-reasons` 렌더(✅/⚠️ 기술 용어) → 구어체 버전으로 교체

### scanner.css
- `.dp-verdict-word`, `.dp-timing-word`, `.dvw-*`, `.dtw-*` 클래스 제거
- `.dp-verdict-poster`, `.dp-timing-mini` 스타일 추가
- `.dp-haiku-secondary` 스타일 추가 (보조 카드용)

## 변경하지 않는 것

- 백엔드 로직, 점수 계산 일절 변경 없음
- conv 계산 로직 변경 없음
- 4축 분석 상세 그리드 변경 없음
- 얼마나?·뭔데?·참고 탭 변경 없음

## 성공 기준

드로워를 열었을 때 살까? 탭에서 그라디언트 판단 포스터가 한줄평과 같은 존재감으로 보이고, 설명이 비투자자도 즉시 이해할 수 있는 구어체로 표시된다.
